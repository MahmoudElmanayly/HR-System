# HR-System

## Steps to run the project:
- Download the repo.
- open the project in vscode.
- go to the **back-end** folder
   - install dependencies ```npm install```
   - run ```npm run start:dev```
- go to the **front-end** folder
   - install dependencies ```npm install```
   - run ```npm run start```
